## Features

* device music play
* background play
* favorite artists
* favorite songs
* playlists
* shortcuts

## Environments

### host

* .NET MAUI
* C# Markup
* C#

### Android

* minSdkVersion 33
* targetSdkVersion 36
* Google Pixel 4a (Android Version 13)
* Google Pixel 7 Pro (Android Version 16)
* Google Pixel Tablet (Android Version 15)

### iOS

* iOS Deployment Target 16
* iPhone 12 mini (iOS 16)
* iPhone 14 Plus (iOS 26)
* iPad Pro (6th generation, iPadOS 18)
